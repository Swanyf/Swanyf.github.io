Array.from()

不是数组原型的方法，只有通过Array调用，可将类数组对象或可遍历的对象转为真正的数组，类数组对象就是拥有 length 属性的对象

操作arguments

Array.of()

直接返回由参数组成的数组数据

find()

find()方法为数组中的每个元素都调用一次函数执行

  • 当数组中的元素在测试条件时返回 true 时, find() 返回符合条件的元素，之后的值不会再调用执行函数。
  • 如果没有符合条件的元素返回 undefined

findIndex()

获取元素索引位置

some()

用于检测数组中的元素是否满足指定条件，直接返回结果为bool值

  • 如果有一个元素满足条件，则表达式返回true , 剩余的元素不会再执行检测。
  • 如果没有满足条件的元素，则返回false。

every()

用于检测数组所有元素是否都符合指定条件

  • 如果数组中检测到有一个元素不满足，则整个表达式返回 false，且剩余的元素不会再进行检测。
  • 如果所有元素都满足条件，则返回 true

promise回调地狱

由于ajax会出现调用时机顺序不同，嵌套的ajax又使代码不整洁。当嵌套的ajax过多时会出现回调地狱。为了解决回调顺序的不确定性以及回调地狱，es6提供了 promise 机制

  • 每个异步任务将返回一个promise对象
  • 每个promise对象都有一个then函数，可以接受两个参数，一个success 处理程序和一个error处理程序
  • 异步任务完成后，函数中的成功或错误处理程序then将只被调用一次
  • 该then函数也将返回一个promise，以允许链接多个调用
  • 每个处理程序（成功或错误）都可以返回一个value，它将argument在promises 链中作为一个传递给下一个函数
  • 如果一个处理程序返回一个promise（发出另一个异步请求），则只有在请求完成后才会调用下一个处理程序（成功或错误）
